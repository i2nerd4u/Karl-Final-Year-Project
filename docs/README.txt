MealMate - Final Year Project

This is my final year project for college. Its a nutrition tracking web app
that uses AWS services (Cognito, DynamoDB, S3, Lambda, API Gateway, Bedrock).

The app lets users track calories, upload food label photos for AI analysis,
get personalised nutrition recommendations, and plan meals.

To run locally just open frontend/index.html in a browser.
The AWS backend needs to be set up separately - see the config files.

You can register an account and log in!
